Permits agricultural buildings to be used for incidental uses accessory to farming and exempts them from certain state structural codes.

[Original Bill Text](https://olis.oregonlegislature.gov/liz/2021R1/Measures/Overview/HB2611)

## Sponsors

**Chief Sponsors:** Vikki Breese-Iverson, David Brock Smith

**Co-Sponsors:** Daniel Bonham, Shelly Boshart Davis, Unknown Legislator, Duane Stark, Tim Knopp, Bobby Levy


## Overview

<i>Prevents application of state structural specialty code to agricultural building used primarily for specified activities.</i>] 
<b>Permits agricultural building to be used for uses other than uses set forth in definition of "agricultural building" if additional uses are incidental and accessory to defined uses, are personal to farm owner and farm owner's immediate family or household and do not pose greater hazard to persons or property than defined uses.</b>
