Adds the Reach Code to the state building code as a specialty code to establish energy efficiency standards for residential and commercial buildings.

[Original Bill Text](https://olis.oregonlegislature.gov/liz/2021R1/Measures/Overview/HB2398)

## Sponsors

**Chief Sponsors:** Ken Helm, Michael Dembrow, Lee Beyer

**Co-Sponsors:** Pam Marsh, Sheri Schouten, Kathleen Taylor


## Overview

<b>Adds Reach Code to state building code as specialty code and gives power of administration and interpretation of Reach Code to Director of Department of Consumer and Business Services.</b> Requires director 	<i>of Department of Consumer and Business Services</i>] to<b> follow same process in adopting or amending Reach Code that director follows in developing residential and commercial building codes and to</b> ensure that statewide Reach Code mandates achievement of not more than 90 percent of site energy use that other statewide residential and commercial building codes require.<b> Requires director to adopt Reach Code not later than October 31, 2021, and at same time director adopts corresponding residential specialty code or corresponding structural specialty code, updating Reach Code at least every three years.</b> 
Permits municipality to adopt Reach Code and require adherence to code as minimum construction standard and method within municipality's jurisdiction<b> notwithstanding requirement that state building code be uniform and applicable to all municipalities in state. Provides that municipality's adoption of Reach Code is not amendment to state building code and does not require approval of director. Provides that municipality that does not adopt Reach Code does not need to enforce Reach Code within municipality's jurisdiction. 
<b>Declares emergency, effective July 1, 2021.</b>
