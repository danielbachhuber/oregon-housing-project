Provides that local governments enforcing wildfire hazard mitigation building code standards are exempt from subsequent updates to the Oregon Residential Specialty Code.

[Original Bill Text](https://olis.oregonlegislature.gov/liz/2023R1/Measures/Overview/SB83)

## Overview

	Provides that local government that adopted and enforces wildfire hazard mitigation building code standards is not subject to any subsequent update to section R327 of Oregon Residential Specialty Code.
