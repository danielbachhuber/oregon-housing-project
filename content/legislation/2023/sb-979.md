## Overview

Provides that constructing child care facility within project for affordable housing does not subject project for affordable housing to prevailing wage requirements if child care facility meets specified requirements. Provides that prevailing wage requirements do apply to construction of child care facility. 
Takes effect on 91st day following adjournment sine die.
