Provides that homesteads rebuilt by the same owner on the same lot after the September 2020 wildfires receive a frozen assessed value based on the 2020-2021 tax year.

[Original Bill Text](https://olis.oregonlegislature.gov/liz/2023R1/Measures/Overview/HB3446)

## Overview

Provides for homestead rebuilt by same owner on same lot to replace homestead destroyed by September 2020 wildfires to have frozen assessed value equal to assessed value for 2020-2021 property tax year to extent of destroyed homestead's square footage. Terminates frozen assessed value for first property tax year that succeeds earlier of date on which property is no longer intended for occupancy as homestead or transfer of ownership of homestead. 
Takes effect on 91st day following adjournment sine die.
