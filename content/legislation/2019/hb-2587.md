Modifies the homestead property tax deferral program to prohibit homes pledged as reverse mortgage security from receiving deferral.

[Original Bill Text](https://olis.oregonlegislature.gov/liz/2019R1/Measures/Overview/HB2587)

## Sponsors

**Chief Sponsors:** Mark Meek


## Overview

<b>For purposes of homestead property tax deferral program for seniors and persons with disabilities, changes prohibition on homestead with outstanding deferred property taxes from being pledged as security for reverse mortgage to prohibition on homestead that is pledged as security for reverse mortgage from being granted deferral under program.</b> Makes exception for certain homesteads 	<i>to prohibition on reverse mortgages for participation in homestead property deferral program for seniors and persons with disabilities</i>]. 
Takes effect on 91st day following adjournment sine die.
